MyFlix API Case Study

Challenge

To fully develop an API for a web application that allows users to create, read, update, and delete information.

Action

I created the business logic, database, and authentication/authorization of the app.

Result

A bolstered understanding of back-end processes and API creation.

MyProcess

Server-Side API

  1. Set up Express to route HTTP requests and defined endpoints
  2. Created a non-relational database using MongoDB
  3. Integrated business logic using Mongoose
  4. Integrated JWT authentication/authentication using Passport
  5. Hosted API on Heroku and connected the database to it
  6. Tested endpoints with Postman
